# Chatting App

Simple echo chatting app between 2 people with E2EE using a shared password.

The client frontend is desgined by ChatGPT.

## Configuration

Run the following command to install the necessary dependancies:

```bash
npm install
```

Refer to [config.json](./config.json) for configuration.

## Extra info

Forward secrecy is non-existant for now. This is a simple chatting app.
